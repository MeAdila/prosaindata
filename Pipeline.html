Tugas Untuk Get data#

  • Untuk membuat pipelines, perlu menggunakan hosting melalui Cloud cluster secara online di https://www.cloudclusters.io/

  • Kemudian login untuk meneruskan pada setiap tools yang dibutuhkan

  • Lakukan “order now” pada tools yang dipilih, lalu akan diteruskan untuk me-manage nama database dan user pada setiap tools

  • Selanjutnya membuat database sesuai dengan perintah yaitu :

    • postgresql local dengan data sepal length

    • postgresql elephant dengan data sepal width

    • mysql local dengan petal length

    • sql server local dengan petal width dan class

  • Setelah data dari beberapa sumber telah dibuat, barulah membuat ‘new pipeline’ pada https://www.orchest.io/

  • Environment disesuaikan dengan menggunakan bahasa pemrograman python dengan memasukan perintah

          pip install pandas scikit-learn matplotlib
          pip install psycopg2-binary
          pip install pymssql
          pip install mysql-connector-python

  • Tahap setelahnya adalah membuat file python berdasarkan data yang diambil dari masing-masing tools sebagai berikut
